🧠 Sistema de Memória

▼

💾 MEMORY.md

Núcleo: Memória curada de longo prazo, insights importantes, decisões e lições aprendidas.

📅 Daily Notes

YYYY-MM-DD.md: Logs diários de atividades, conversas e contexto recente.

📂 Topic Files

Organizados: Projetos específicos, lessons learned, decisões importantes por categoria.

🔄 Consolidação

Fluxo: Daily notes → revisão periódica → destilação para MEMORY.md → contexto persistente.

Como funciona:

Informações fluem dos daily notes para o núcleo de memória através de revisão e curadoria automática durante heartbeats. Isso garante contexto rico sem perder informações importantes.

💓 Heartbeat & Automação

▼

💓 Health Checks

Monitora sistemas, APIs, conexões. Alertas automáticos em caso de problemas.

🗜️ Memory Compaction

Consolida daily notes, remove duplicatas, mantém MEMORY.md otimizada.

📊 Pool Monitoring

Acompanha posições DeFi, yields, oportunidades de rebalancing.

📧 Smart Briefing

Resume emails importantes, agenda do dia, market updates relevantes.

🚨 Alert System

Notificações inteligentes: price alerts, calendar events, mentions importantes.

🔄 Sync & Backup

Sincroniza arquivos importantes, backup de configurações, git commits automáticos.

⏰ Scheduling:

Heartbeats: Checks periódicos (~30min) para tarefas batch
Cron Jobs: Timing exato para tarefas específicas
Event-driven: Reação a triggers externos (emails, mentions, price changes)
